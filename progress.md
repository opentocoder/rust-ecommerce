## 运行说明

### 启动后端
```bash
cargo run -p backend
```

### 种子数据
```bash
cargo run -p backend --bin seed
```

### 启动前端 (需要安装 trunk)
```bash
cargo install trunk
cd frontend && trunk serve
```

### 访问
- Backend API: http://localhost:3000
- Frontend: http://localhost:8080
